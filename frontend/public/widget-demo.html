Add this widget to your website by including these lines before the closing </body> tag:

<!-- CallCenter AI Chat Widget --> <script src="http://138.2.68.107/chat-widget.js"></script> <script> CallCenterChat.init({ apiUrl: 'http://138.2.68.107:8443', tenantId: 'demo-tenant', position: 'bottom-right', // or 'bottom-left' primaryColor: '#4F46E5', title: 'Chat with us', subtitle: 'We typically reply instantly', welcomeMessage: 'Hi! How can I help you today?' }); </script>

Configuration Options

  • apiUrl (required): Your CallCenter API URL
  • tenantId (required): Your tenant identifier
  • position: 'bottom-right' or 'bottom-left' (default: 'bottom-right')
  • primaryColor: Hex color code (default: '#4F46E5')
  • title: Widget header title
  • subtitle: Widget header subtitle
  • welcomeMessage: First message displayed to users
